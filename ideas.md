# 「大林慢遊：銅板系生存指南」設計規劃

## 選定設計方向：復古懷舊 × 青年文化混搭

### 設計運動 (Design Movement)
**新懷舊主義 (Neo-Nostalgia)** - 融合 1960-1980 年代台灣鐵道文化與當代大學生網路文化的混搭美學。不是純粹懷舊，而是用現代設計語言詮釋過去，創造時間穿越感。

### 核心原則 (Core Principles)

1. **時間層疊** - 頁面視覺上呈現多個時代的痕跡：大林糖廠的工業遺跡、日式建築、當代大學生的迷因文化
2. **預算友善的視覺** - 設計本身要傳達「少花錢也能有好體驗」的理念，使用粗糙邊界、手繪感、不完美美學
3. **互動即敘事** - 每個互動工具（轉盤、試算機、時間軸）都是故事的一部分，而非獨立功能
4. **年輕人的語言** - 視覺上要有迷因感、玩樂感，文案要有梗，排版要有呼吸感

### 色彩哲學 (Color Philosophy)

| 色彩 | 用途 | 情感意義 |
|------|------|--------|
| **大林糖廠黃** (#F4D03F / #FFC107) | 主色，CTA 按鈕、重點標題 | 溫暖、懷舊、希望、青春 |
| **鐵道灰** (#5A5A5A / #6B7280) | 次色，副標題、邊框、分隔線 | 沈穩、工業感、時間感 |
| **青綠** (#4A9B6F / #10B981) | 點綴色，成功狀態、自然感 | 生機、大林自然景觀、療癒 |
| **米白** (#F5F1E8 / #FFFBF0) | 背景色 | 紙質感、懷舊、溫暖 |
| **深灰** (#2C2C2C) | 文字主色 | 易讀性、沈穩 |

**色彩邏輯**：
- 背景用米白營造紙質感（像是舊火車票或糖廠宣傳單）
- 黃色用於吸引注意力（轉盤、試算結果、重點 CTA）
- 灰色用於建立結構感和時間感
- 青綠用於正面反饋和自然場景

### 版面範例 (Layout Paradigm)

**長滾動 + 分段式設計**：
- 每個功能區塊獨立成段，用 SVG 分隔符號連接（波浪、齒輪、火車軌道圖案）
- 不是中心對齐的呆板排列，而是左右交替、錯落有致的非對稱佈局
- 使用 CSS clip-path 製造斜邊、對角線切割，製造動感
- 大量留白，讓內容「呼吸」

### 簽名元素 (Signature Elements)

1. **手繪邊框** - 所有卡片、按鈕用手繪風格邊框（SVG 或 CSS border-radius 不規則變化）
2. **火車軌道圖案** - 用於分隔符、進度指示器、時間軸
3. **復古郵票效果** - 重要資訊用郵票邊框呈現（虛線邊框 + 黃色背景）

### 互動哲學 (Interaction Philosophy)

- **轉盤旋轉** - 要有物理感，轉動時有慣性、減速效果
- **試算機輸入** - 即時反饋，數字變化時有數字滾動動畫
- **卡片展開** - 點擊時卡片展開、陰影加深，營造 3D 浮起感
- **時間軸滑動** - 滑到該時段時自動高亮，有「進度」感

### 動畫指南 (Animation)

- **進入動畫**：淡入 + 向上滑動（300ms，ease-out）
- **懸停效果**：按鈕/卡片輕微放大（1.05x）+ 陰影加深
- **轉盤旋轉**：快速旋轉 → 逐漸減速 → 停止（cubic-bezier 模擬物理）
- **數字跳動**：試算結果數字從 0 滾動到最終值（800ms）
- **時間軸進度**：當前時段卡片有脈衝動畫（pulse effect）

### 排版系統 (Typography System)

| 用途 | 字體 | 大小 | 粗細 | 用例 |
|------|------|------|------|------|
| **大標題** | Noto Sans TC Bold + Zen Maru Gothic | 48px | 700 | 首頁標題「大林慢遊」 |
| **中標題** | Noto Sans TC SemiBold | 28px | 600 | 各功能區塊標題 |
| **小標題** | Noto Sans TC Medium | 18px | 500 | 卡片標題、時間軸時間 |
| **正文** | Noto Sans TC Regular | 16px | 400 | 描述文字、美食介紹 |
| **輔助文字** | Noto Sans TC Regular | 14px | 400 | 提示、標籤、價格 |
| **迷因文案** | Zen Maru Gothic | 20px | 700 | 「窮學生救星」、「警告！」 |

**排版邏輯**：
- 標題用 Zen Maru Gothic（手寫感）製造年輕、玩樂的氛圍
- 正文用 Noto Sans TC 保持易讀性
- 行高 1.6-1.8 營造寬鬆感
- 字母間距微調，製造精緻感

## 設計決策依據

這個設計方向選擇的理由：

1. **解決使用者痛點** - 視覺上要傳達「少花錢也能有好體驗」，所以不用高級感的設計，而是用「粗糙美」、「不完美美」
2. **年輕人的語言** - 迷因感、手寫感、玩樂感是大學生的視覺語言
3. **地方特色融合** - 大林糖廠黃 + 鐵道灰 + 青綠不是隨意選擇，而是大林本身的視覺特徵
4. **互動即故事** - 每個功能都是一個故事片段，而非孤立的工具
